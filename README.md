# PDF AI Pipeline

A simple Python-based pipeline to ingest one or multiple PDF documents and:

- Extract text and tables
- Preserve document structure (sections)
- Summarize key content using an LLM
- Answer natural language questions using extracted content

## Features

- Works with single or multiple PDFs
- Table-aware summarization
- No vector databases or complex infrastructure
- Single-file, easy-to-understand implementation

## Installation

```bash
pip install -r requirements.txt
```
